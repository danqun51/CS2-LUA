# CS2-LUA

版本：`1.0.1`  
更新时间：`2026-07-15`  
运行时：`LuaJIT 2.1 / Lua 5.1 语义`  
脚本目录：`Counter-Strike Global Offensive\game\bin\win64\lua`  
菜单热键：`Home`

# 使用方法

打开exe文件直接注入，或者用自己的注入器注入dll文件

遇到无法注入问题可以设置cs2启动项为`-allow_third_party_software`
