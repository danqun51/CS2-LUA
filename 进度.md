## utils API 状态

参考：https://docs-csgo.neverlose.cc/documentation/variables/utils

### 已实现

- `utils.execute_after(delay, callback, ...)`
- `utils.console_exec(cmd, ...)`：通过 `engine2.dll!InputService_001` vtable[25]
- `utils.opcode_scan(module, signature[, offset])`
- `utils.create_interface(module, interface)`
- `utils.random_int(min, max)`
- `utils.random_float(min, max)`
- `utils.random_seed(seed)`
- `utils.get_netvar_offset(table, prop)`：通过当前 offsets 表查询，非完整 RecvTable/Schema 实现

### 已提供兼容返回结构，但尚未连接 CS2 原生追踪

- `utils.trace_line(...)`
- `utils.trace_hull(...)`
- `utils.trace_bullet(...)`
- `utils.net_channel()` 及 NetChannel 字段/方法

这些函数现在不会崩溃脚本，并返回与文档相近的 table 结构；trace 默认表示无遮挡，NetChannel 默认返回本地占位值。

### 待实现

- 原生 TraceRay / TraceShape
- 原生 bullet penetration/damage
- 原生 NetChannelInfo
- `get_vfunc`：Lua 5.4 没有 LuaJIT FFI，需要设计安全的 native call wrapper
